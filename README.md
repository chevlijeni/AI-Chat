# Jeni AI - Backend Developer Assistant

A complete, full-screen AI chat application built with **Node.js, Express, and Vanilla JavaScript/CSS**. This project leverages the powerful **Google Gemini API** (`gemini-2.0-flash-lite`) to provide instant, intelligent conversational responses while automatically managing chat states.

## Features
- **Local History Memory:** Automatically saves your active chat sessions and individual chat messages in your browser's `localStorage` so they persist through page reloads.
- **Multiple Iterative Chats:** Allows you to generate multiple isolated conversation threads, instantly switch between them using the sidebar, or delete past histories.
- **Auto-Title Generation:** Automatically detects your first message to dynamically read and generate an intelligent title for your sidebar session.
- **Markdown Rendering:** Fully supports and beautifully parses markdown generated by the AI directly in the chat bubbles via the `marked.js` library.
- **Graceful Error Handling:** Specially designed error interception for Google SDK Rate Limit and Quota Exchaustion warnings.

## Prerequisites
Before you begin, make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher)
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

---

## 🚀 Installation & Setup

**1. Clone the repository**
```bash
git clone https://github.com/chevlijeni/AI-Chat.git
cd AI-Chat
```

**2. Install dependencies**
Install the necessary Node modules (express, dotenv, @google/generative-ai).
```bash
npm install
```

**3. Configure Environment Variables**
Copy the included template file to create your functional `.env` file!
```bash
cp .env.example .env
```
Open the `.env` file in your code editor and paste your Gemini API Key:
```env
PORT=3000
GEMINI_API_KEY=your_actual_api_key_goes_here
```

**4. Start the Server**
Run the backend Node server securely.
```bash
node server.js
```

**5. Start Chatting!**
Open your favorite web browser and navigate to:
**[http://localhost:3000](http://localhost:3000)**
